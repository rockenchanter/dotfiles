# Overview

Dotfiles for my Void Linux setup. Everything is based on
gruvbox colorscheme. I'm using Wayland as display server, but you can also run 
Xorg based applications with xwayland

Applications:
- window manager: *sway*
- status bar: *waybar*
- terminal emulator: *foot*
- text editor: *neovim*
- music player: *mpd + mpc*
- video player: *mpv*
